変数

c言語では変数には型があります.変数とは数学で言うところのXに該当する部分です.

  • char
    size1バイト
  • int
    size4バイト
  • long
    size8バイト
  • float
    size8バイト
  • double
    size8バイト

これらの演算は演算子を使用することで計算することができます

          
            int a = 2;
            int b = 3;
            int c = a*b;//cの中身は6です.

またユーザー定義型もあります.
ユーザー定義型は使用者が定義し,通常の型として利用できます.

  • struct
  • enum
          
            struct Parson{
              char* str;//名前
              int old;
            };
            struct Parson kosei;
            kosei.str = "kosei";
            kosei.old = 19;
