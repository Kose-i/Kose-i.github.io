型

型とは,メモリのサイズを決めるためのものです.

intのサイズが知りたいときは,sizeof演算子を使用して,sizeof(int) とすることでバイト数を知ることができます.

voidはデータを持たないため,関数に主に使用されます.

charは文字を表すときに使用されます.

int,short,longは整数を表すときに使用されます.

signed,unsignedは符号付き(signed)符号無し(unsigned)を選択できる修飾子です.

static,constは変化せず,定数とする修飾子です.

externは他のファイルからの宣言を読み込むことができます.

typedef を使用することでエイリアス(別名)をかけることができます.

    char char は サイズが 1バイト 文字を表現するのに使います
    int int は サイズが 4バイト 整数を表現するのに使います
    short short は サイズが 4バイト 整数を表現するのに使います
    long long は サイズが 8バイト 大きい整数を表現するのに使います
    float float は サイズが 4バイト 小数を表現するのに使います
    double double は サイズが 8バイト 小数を表現するのに使います

これらの演算は演算子(後述)を使用することで計算することができます

          
            int a;//宣言
            a = 2;//初期化
            int b = 3;//宣言と初期化
            int c = a+b;//cの中身は5です.
            int c = a-b;//cの中身は-1です.
            int c = a*b;//cの中身は6です.
            int c = a/b;//cの中身は0です. ２÷３は０あまり２です.そのため整数では０となってしまいます.
            int c = a%b;//cの中身は2です. ２÷３のあまりが来ると考えてください.

またユーザー定義型struct enumもあります.
ユーザー定義型は使用者が定義し,通常の型として利用できます.

これらの演算ではそれぞれの要素にしか演算子を使用することはできません

詳しくは構造体についてを参照してください.

          
            struct Parson{
              char* str;//名前
              int old;//年齢
            };//struct の宣言

            struct Parson kosei;//変数の宣言
            kosei.str = "kosei";//それぞれの値の初期化
            kosei.old = 19;//それぞれの値の初期化

変数

c言語では変数には型があります.
変数とは数学で言うところのXに該当する部分です.

[型] [変数名];

として宣言します.

関数

関数とは数学で言うところのy=f(x)のｆに該当する部分です

型を持ち,引数を取り,戻り値を返します.

実際に例を見てみると,

          
            int funtion(int x) {
              return (x + 1);
            }

この関数の型はint型で,ここでの引数はint で,返り値は,その数に1を足した数です.

          
            int t = 3;
            t = funtion(t);//tは4になる.

void print(char str[])は関数であることは分かりました.

if else文

条件に適うかどうかで実行するかしないかを決定できます.

if のあとにelse if,elseが実行されます.

          
            if (1) {
              func();//必ず実行される.
            }
            if (0) {
              func();//必ず実行されない.
            } else {
              func();//必ず実行される.
            }

switch文

switch文は,条件を多分岐に指定するときに使用します.

breakを指定しなければその後もすべて実行されます.

          
            int tmp = 3;
            switch(tmp) {
              case 1://if (tmp == 1)
                func();//実行されない
                break;
              case 2://if (tmp == 2)
                func();//実行されない
                break;
              case 3://if (tmp == 3)
                func();//実行される
              case 4://if (tmp == 4 || tmp == 3)
                func();//実行される
              default://else
                func();//実行される
            }

for文

for 文は繰り返し処理用の構文です.

for (/*初期化*/;/*継続条件*/;/*毎回実行処理*/)

          
            for (int i = 0;i != 5;++i) {
              printf("%d", i);
            }

while文

while(/*継続条件*/)

継続条件がtrueのとき処理は繰り返し実行されます.

do-while文

do{}while(/*継続条件*/);

継続条件がtrueのとき処理は繰り返し実行されます.

while文との違いは必ず一回は実行される点です.

マクロ

#includeと指定することでファイル内の情報を書き込むことができます.

そのため,ヘッダファイルを作成するときはインクルードガードを使用しなければなりません.関数の二重定義となり,エラーとなります.

#defineと上部で宣言して値を書き込むことでその後のその変数にはその値が自動的に挿入されます.

          
            #define SIZE 256
            char t[SIZE];//t[256]と同じ宣言になる

また分割コンパイル時のインクルードガードを使用するときにも使用します

          
--func.h--

          
            #ifndef compile_guard//define されているかを調べる.
            #define compile_guard
            void func();
            #endif
