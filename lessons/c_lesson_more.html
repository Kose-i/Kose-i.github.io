構造体について

構造体はメンバー変数を持つ集合体です.

構造体はメンバー変数としてなどの int char 型の変数を持つことができます.

typedefを使用することで一般の方と同じようにstructなしで使用することもできます.

構造体はポインタを経由することでアロー演算子を使用して各メンバにアクセスすることができます

以下がテストコードです

        
        struct parson{
          char* name;
          unsigned int old;
        };

        typedef struct Tmp{
          int tm;
        }Tmp;
        
        int main(void) {
          struct parson kosei;
          kosei.name = "tamura";
          kosei.old = 19;
          Tmp* tmp;
          tmp->tm = 2;
        }
