C言語とC++言語との大きな違いはクラスが導入されたことです.

C++言語では,オブジェクト指向プログラミング(Object Oriented Programing)をします.

抽象化して使い回しををすることでコードを短くしバグの発生を抑えましょう.

        
          class parson{
            private:
              int credit_card_id;//クレジットカード番号
              int address;//アドレス
            public:
              int name;//あなたの名前
              int old;//あなたの年齢
          };

private,publicというものがあります.

private (プライベート),つまり他人には公開されません.

一方, public とは他人に公開されます.

クラスの外からでもアクセスできるものはpublicな情報であり,

クラスの外からアクセスできないものはprivateな情報となるわけです.

privateにすることで,気づかない間に値が変わるということを防ぐことができます.


次は,関数を入れてみます.

        
          class parson{
            private:
              int credit_card_id;
              int address;
            public:
              int name;
              int old;
              void print_name(){
                printf("%d\n", name);
              }
              void change_address(int new_address){
                address = new_address;
              }
          };

このように,クラス内の関数はクラス内のどのpublic privateな値でも読み書きすることができます.

クラスは継承をすることができます.

protected識別子を用います.

Template

templateは抽象化して,具体化するという過程を踏んでいます.

typeを設定するのを省き,typeを動的に決定します.

typeを特別に設定することが可能です.

        
          #include < iostream >
          
          template< typename T1 >
          void print(const T1& print_target)
          {
            std::cout << print_target << '\n';
          }
          void print(const int& print_target)
          {
            std::cout << "int:" << print_target << '\n';
          }
          
          int main(int argc, char** argv)
          {
            std::string str{"str"};
            print(str);
            auto temp = 2;
            print(temp);
          }
