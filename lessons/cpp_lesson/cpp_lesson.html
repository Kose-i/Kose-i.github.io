C言語とC++言語との大きな違いはクラスが導入されたことです.

C++言語では,オブジェクト指向プログラミングをします.

抽象化して使い回しををすることでコードを短くしバグの発生を抑えましょう.

自分も誤解や理解が不十分なこともあるかと思いますが,お付き合いください.

        
          class parson{
            private:
              int credit_card_id;//クレジットカード番号
              int address;//アドレス
            public:
              int name;//あなたの名前
              int old;//あなたの年齢
          };

private,publicというものがあります.

private (プライベート),つまり他人には公開されません.

一方, public とは他人に公開されます.

クラスの外からでもアクセスできるものはpublicな情報であり,

クラスの外からアクセスできないものはprivateな情報となるわけです.

privateにすることで,気づかない間に値が変わるということを防ぐことができます.


次は,関数を入れてみます.

        
          class parson{
            private:
              int credit_card_id;
              int address;
            public:
              int name;
              int old;
              void print_name(){
                printf("%d\n", name);
              }
              void change_address(int new_address){
                address = new_address;
              }
          };

このように,クラス内の関数はクラス内のどのpublic privateな値でも読み書きすることができます.
